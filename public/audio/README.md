# Audio local (100% sin conexión)

Coloca aquí los archivos MP3/OGG (mínimo 64 kbps, máximo 100 kB por palabra).

Convenciones de nombre que espera la aplicación:

- Palabras de vocabulario: `audio/words/<wordId>.mp3`
  - Ejemplo: `audio/words/w-namaste.mp3`
- Ejemplos de gramática: `audio/grammar/<topicId>-s<paso>-e<ejemplo>.mp3`
  - Ejemplo: `audio/grammar/g-a1-pronombres-s1-e2.mp3`

Si un archivo falta o falla, la interfaz muestra un indicador de error con
opción de reintento; en gramática se muestra además la transcripción
fonética (IPA aproximada) como alternativa.
